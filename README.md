Procesamiento Digital de Imágenes con OpenCV aplicado a mi gato 'lulú'

Este proyecto forma parte de mi especialización en Inteligencia Artificial y Visión Artificial.
Utilicé este notebook para experimentar con técnicas fundamentales de manipulación de píxeles y espacios de color.

Tecnologías utilizadas

OpenCV: Para la manipulación de la imagen y transformaciones.
NumPy: Para el manejo de las matrices de la imagen.
Matplotlib: Para la visualización y comparación de resultados.

Conceptos Aplicados
Espacios de Color: Conversión de BGR a RGB para visualización correcta.
Transformaciones Lineales: Ajuste manual de brillo y contraste.
Manipulación de Canales: Ajuste de saturación en el modelo de color.
Anotación de Imágenes: Inserción de formas y texto técnico sobre la matriz de la imagen.

Resultado
El proyecto toma una imagen original (mi gato) y aplica un pipeline de procesamiento para mejorar la visibilidad y resaltar características específicas.
